# v0.2.0-notes-apply-asset-setup

Release boundary: controlled notes/status/timeline apply workflow plus optional asset setup. RDF/export is not part of this release.

## Added

- Added `Setup Notes Workflow` to create the `Planning Review` notes columns and `Decision Staging` table.
- Moved the maintained ApplyNotes Office Script into `office-scripts/apply_notes.ts`.
- Added `office-scripts/apply_asset_mappings.ts` as a starter controlled-write script for asset mappings.
- Added `Setup Asset Workflow` as an optional add-in button.
- Added public-safe starter TSVs for decision staging and asset setup tables.
- Added `modules/assets.formula.txt` with dynamic-array asset review queues.

## Deferred

- RDF/export
- SHACL validation
- Power Query bridge
- Ontology publication
- Generated workbook artifacts

## Impact

- Workbook behavior: new add-in setup surfaces for notes workflow and optional asset setup.
- Formula logic: existing capital planning formulas are unchanged; new asset formulas are review-only.
- Main report totals: no intended change.
- Subtotal flags: no intended change.
- Cap remaining values: no intended change.
